From the Back-End to the Front-End

Learning Web Development

By Abner Vinaja

01/25/2021

I began to learn web development back in the Fall 2020 semester in my Web Technologies course. Then recently, I have begun to further strengthen my skills through Udacity's Front End Web Developer Nanodegree. Though each gave a different approach to learning web development, both have helped me to develop my skills.

HTML and CSS

The first things that I learned in my Web Technologies course were HTML and CSS. I learned how to create basic static web pages using these two tools. I used HTML elements like tables and CSS properties like float to structure these simple pages.

PHP

I then learned how to use PHP as a back-end programming language that would generate the HTML for my webpages, and make my webpages more dynamic. Using a VM from Amazon Lightsail, I installed MySQL and a PHP interpreter to run my PHP code and to connect it to database tables. I learned how to use PHP to carry out actions like submitting forms, redirection, and basic authentication.

Front End Web Developer Nanodegree

Now, I am currently taking Udacity's Front End Web Developer Nanodegree to begin my professional path as a Web Developer. As of right now, I have strengthed my use of CSS thanks to the introduction of CSS Flexbox and CSS Grid. I look forward to learning how to use Javascript to add functionality to my webpages.
