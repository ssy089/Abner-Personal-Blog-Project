Understanding and Using Databases

By Abner Vinaja

01/25/2021

From the first day of my Database Systems course, my professor made it clear that he wanted to understand the implementation as well as the use of databases. This involved not only learning SQL, but also understanding how files are structured and how database efficiency is achieved.

File Structures

To understand databases, we first reviewed how files are structured and managed. We paid particular attention to how database queries with multi-value keys are processed. We covered concepts like B+ trees, inverted lists, and hash tables.

SQL

Next, we implemented SQL, particularly using MySQL. I learned how to create databases and tables with their respective permissions, and how to carry out queries. We also practiced relational algebra and relational calculus, which helped us to better understand complex queries. I myself have learned how to use SQL with Java, Python, and PHP.

Implementation Issues

At the end of the course, we reviewed some implementation issues that database system mangagers have to deal with. These included integrity, query optimization, and concurrency control.
